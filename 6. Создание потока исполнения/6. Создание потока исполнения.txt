~~~~~~~~~~~~~~~~~~~~~~~~~СОЗДАНИЕ ПОТОКА ИСПОЛНЕНИЯ~~~~~~~~~~~~~~~~~~~~~~~~~

 Для создания потока нужно получить объект типа Thread
  1. Реализовать интерфейс Runnable
  2. Расширив класс Thread

I. Реализация интерфейса Runnable

 Для реализации интерфейса Runnable в классу необходимо объявить 
единственный метод run():
	public void run()

 В методе run() можно вызывать другие методы, но run() - единственный вход
в параллельный поток.
 После создания класса в этом классе нужно получить экземпляр класса Thread:
	Thread(Runnable объектПотока, String имяПотока)
- это конструктор этого объекта

 После того, как поток создан, он не запускается, пока не будет вызван 
метод start() (Смотри Example6.1)

 Здесь в конструкторе NewThread() создается новый объект класса Thread
 Передача ссылки this означает, что при запуске потока t будет выполняться
метод run() из класса NewThread
 Метод start() запускает метод run(), где начинается цикл for
 После вызова метода start() Класс NewThread возвращает управление 
методу main(), где начинается свой цикл for
 Результат выполнения программы:
	Дочерний поток: Тhrеаd[DemoThread, 5, main]
	Главный поток: 5
	Дочерний поток: 5
	Дочерний поток: 4
	Главный поток: 4
	Дочерний поток: 3
	Дочерний поток: 2
	Главный поток: 3
	Дочерний поток: 1
	Дочерний поток завершен
	Главный поток: 2
	Главный поток: 1
	Главный поток завершен

2. Расширение класса Thread

 Для расширения класса Thread необходимо переопределить метод run()
 Также в этом классе должен быть вызван метод start() (смотри Example6.2)
 
 Здесь метод super() вызывает конструктор класса Thread с одним параметром -
 именем потока:
	public Thread(String имяПотока)
 Результат этой программы тот же, что и у первой.
 
Выбор метода для создания потока

 В классе Thread определяется ряд методов, которые могут быть переопределены
в производных классах,
из них обязан быть переопределен только сетод run()
 Если нужно реализовать только метод run(), то тогда проще использовать
интерфейс Runnable
