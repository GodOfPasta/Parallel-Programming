~~~~~~~~~~~~~~~~~~~КЛАСС THREAD И ИНТЕРФЕЙС RUNNABLE~~~~~~~~~~~~~~~~~~~~

 Чтобы создать новый поток исполнения, нужно расширить класс
Thread или реализовать интерфейс Runnable
 В классе Thread определяется ряд методов, помогающих упрвлять 
потоками исполнения:
	getName		- получает имя потока исполнения
	getPriority	- получает приоритет потока исполнения
	isAlive		- Определяет, исполняется ли поток
	join		- Ожидает завершения потока исполнения
	run		- Задает точку входа в поток исполнения
	sleep		- Приостанавливает поток на заданное время
	start		- Запускает поток, вызывая метод run()
