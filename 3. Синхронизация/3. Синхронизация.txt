~~~~~~~~~~~~~~~~~~~~~~~~~~~СИНХРОНИЗАЦИЯ~~~~~~~~~~~~~~~~~~~~~~~~~~

 Для синхронизации потоков в Java используется /монитор/
 Как только поток исполнения входит в монитор, все другие потоки исполнения ждут, пока тот не покинет монитор
 У каждого объекта имеется неявный монитор, вход в который осуществляется автоматически, когда для него вызывается синхронизированный метод
 Для одного объекта нельзя вызвать два синхронизированных метода
