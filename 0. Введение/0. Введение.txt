~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ВВЕДЕНИЕ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  В Java многопоточность встроена.
  Многопоточная программа состоит по крайней мере из двух параллельно 
выполняемых частей.
  Каждая часть - поток исполнения.
  Многозадачность задается двумя видами:

На основе процессов		 |  На основе потоков
  -Работает в ОС		 |    -Работает в программах
  -Позволяет запускать 		 |    -Позволяет программе выполнять 
  >2 программ			 |    >2 задач
  -По сути - много больших	 |    -Небольшие задачи, использующие
  задач, для каждой из которых	 |    одно адресное прост-во
  требуется свое адресное	 |    -Связь между потоками, как и 
  прост-тво			 |    переключение между ними дешево
  -Переключение между процессами |
  дорого			 |
  
  Многопоточность позволяет по максимуму загрузить вычислительные
мощности.
  Многопоточность сводит к минимуму время ожидания:
    Скорость передачи данных << Скорость обработки
    => можно выполнять что-то другое во время передачи
